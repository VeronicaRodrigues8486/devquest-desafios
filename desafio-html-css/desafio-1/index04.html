Lista de Definições

HTML
linguagem de marcação de texto para criação de conteúdos de um site.
CSS
Linguagem de marcação para a criação de designg de um site.
Git
Sistema de controle de versão.
Gitgub
Plataforme web onde podemos criar projetos dentro de repositórios e compartilhar isso com outras pessoas.
JavaScript
JavaScript é uma linguagem de programação que possibilita a implementação de objetos complexos em páginas web.
